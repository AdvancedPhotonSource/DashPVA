Workbench Viewer Documentation

Overview and usage notes for the Workbench.

The Workbench viewer provides 2D, 3D, and 1D visualization of HDF5 datasets along with tools for ROI and speckle analysis.

Features

  • Load single HDF5 files or entire folders
  • Visualize datasets in 2D and 3D (when PyVista is available)
  • ROI creation and statistics
  • ROI Calculator (A − B), ROI Plot, and ROI Math tools
  • Playback of 3D stacks with FPS control

How to open this documentation

Use the menu Documentation → Open Documentation or press F1 within the Workbench window.

Regions of Interest (ROIs)

ROI Basics

  • Draw ROIs using the Draw ROI action in the 2D viewer. Drag to move; drag handles to resize. A rotate handle allows rotation about the center.
  • ROI statistics (sum, min, max, mean, std, count, x, y, w, h) appear in the ROI dock. Right-click an ROI for actions such as Show Stats, Rename, Set Active, Open ROI Plot, Delete, and Save ROI.
  • Use the checkbox in the ROI stats table to show/hide the ROI name label over the image.

ROI Sources

  • Memory: ROIs drawn in the current session and associated with the currently viewed dataset.
  • File: ROIs loaded from HDF5 metadata. The ROI Calculator UI shows options to select from file and to import ROIs; these file-based features are placeholders and not wired yet. Current calculations use in-memory ROIs only.

ROI Calculator Operations

  • Average Series: A − B: Produces a 2D mean-difference image: mean(A) − mean(B) over frames. Also computes and plots a per-frame series diff = mean(Aframe) − mean(Bframe). On a single-image dataset, this produces a single value.
  • Pixel-wise: A − B: Produces a 2D image or a 3D stack where each frame is A − B, aligned frame-by-frame.
  • Custom Expression: UI placeholder; currently disabled.

Matching & Alignment

  • Strict Match on: A and B must have identical shapes (including frame count). If shapes differ, computation is aborted with a clear message.
  • Strict Match off + Auto-Intersect on: A and B are truncated/cropped to the common overlapping region: T = min(frames), H = min(height), W = min(width). Notes about cropping/truncation appear in the status bar.
  • Strict Match off + Auto-Intersect off: Shapes must still match; otherwise, computation is aborted.

Time Axis and Include ROI Frames

These controls are present for future functionality and are not currently applied to computations.

Results

  • Series tab: Plots per-frame mean(A) − mean(B) and shows a metrics table: frames, ROI pixel count, mean(A), mean(B), mean/min/max/std of the diff. For single-image data, the plot shows a single point.
  • Image tab: Shows the computed image/stack result depending on the selected operation (Average Series image or Pixel-wise stack).
  • Send to 2D Workspace: Sends the current result image/stack to the main 2D workspace for further analysis.
  • Save As HDF5: Saves the result as /entry/data/result in the chosen .h5/.hdf5 file. Dataset attributes include: operation, roiA_name, roiB_name, strict, auto_intersect, and notes (if any).

ROI Plot and ROI Math

  • ROI Plot (dock/window): Flattens the ROI sub-image to a 1D vector and plots Index vs Value. Supports adding a docked view or a modeless window.
  • ROI Math dock: Define expressions over the flattened ROI data (x: indices, y: ROI values, np: numpy). Multiple expressions render as separate curves.

Saving ROIs to HDF5

ROIs can be saved under /entry/data/rois/<name> with attributes x, y, w, h, and source_path. The dataset contains either a 2D ROI image or a 3D stack of ROI frames when the source is a time series.

Technical details (advanced)

  • extract_roi_stack(main_window, roi) (utils/roi_ops.py): For 3D data, returns a stack with shape (T, H, W). For a single 2D frame, returns (H, W). Uses transform-aware extraction via roi.getArrayRegion(frame, image_item) when available, falling back to integer pixel slicing via roi.pos()/roi.size(). If an extraction yields an empty area, falls back to zeros of the current ROI box size for consistency. Across frames, trims to the smallest extracted height/width to ensure a stackable array. Returns np.float32 arrays.
  • align_stacks(A, B, strict=True, auto_intersect=False) (utils/roi_ops.py): Normalizes inputs so 2D becomes (1, H, W). With strict=True, shapes must match exactly. With auto_intersect, crops/truncates both A and B to their common (T, H, W) and records notes. Otherwise requires shapes to match. Returns (A_aligned, B_aligned, info) where info.ok indicates success and info.notes lists alignment actions.
  • per_frame_mean(stack) (utils/roi_ops.py): Returns a 1D np.float32 array of per-frame means (length T). For a single 2D image, returns length 1.

Tips & Notes

  • For each frame, the value in the Average Series plot equals the average of the pixel-wise A − B within that frame.
  • If either ROI is partially outside the image, empty regions are filled with zeros for the purposes of series computation.
  • File-based ROI selection and the ROI Calculator "Custom Expression" operation are planned features and currently disabled.
